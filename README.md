## Использование:
2. В проекте реализованы операции:

### 2.1. Страница «События»:
2.1.1. Реализован набор функций и главная функция, принимающая на вход строку с датой и второй необязательный параметр —
диапазон данных. По умолчанию диапазон равен одному месяцу (с начала месяца, на который выпадает дата, по саму дату).

Возможные значения второго необязательного параметра:
W — неделя, на которую приходится дата;
M — месяц, на который приходится дата;
Y — год, на который приходится дата;
ALL — все данные до указанной даты.

Возвращаемый JSON-ответ содержит следующие данные:

2.1.2. «Расходы»:
Общая сумма расходов.
Раздел «Основные», в котором траты по категориям отсортированы по убыванию. 
Данные предоставляются по 7 категориям с наибольшими тратами, 
траты по остальным категориям суммируются и попадают в категорию «Остальное».
Раздел «Переводы и наличные», в котором сумма по категориям «Наличные» и «Переводы» отсортирована по убыванию.
«Поступления»:
Общая сумма поступлений.
Раздел «Основные», в котором поступления по категориям отсортированы по убыванию.

2.1.3. Курс валют.
Наименования валют по умолчанию - "USD", "EUR"
При необходимости состав списка валют может быть изменен в файле 'data/user_settings.json', 
для этого необходимо изменить список "user_currencies" в указанном файле

Курсы валют будут получены путем запроса на API-сервер https://v6.exchangerate-api.com/
Ключ для генерации запроса нужно указать в файле env.example

2.1.4. Стоимость акций из S&P 500.
Наименования котируемых акций по умолчанию - "AAPL", "AMZN", "GOOGL", "MSFT", "TSLA"
При необходимости состав списка акций может быть изменен в файле 'data/user_settings.json', 
для этого необходимо изменить список "user_stocks" в указанном файле

Котировки акций будут получены путем запроса на API-сервер https://api.twelvedata.com
Ключ для генерации запроса нужно указать в файле env.example

2.1.4.1 **ВАЖНО !!!** 
При доступе в бесплатном режиме указанный API-сервер отвечает не более чем на 8 запросов в 1 минуту,
поэтому для корректной работы программы в режиме _**бесплатного доступа**_ 
_ЗАПРЕЩЕНО_ устанавливать в списке "user_stocks" более 8 наименований,
также **_крайне не рекомендуется**_ повторно запускать исполняемый код в течение 1 минуты с первого запуска

### 2.2 Сервис "Поиск переводов физическим лицам"
2.2.1 Функция возвращает JSON со всеми транзакциями, которые относятся к переводам физлицам.
Категория такой транзакции — Переводы, а в описании есть имя и первая буква фамилии с точкой.

### 2.3 Отчет "Траты по дням недели"
2.3.1 Функция принимает на вход:
- датафрейм с транзакциями,
- опциональную дату.
Если дата не передана, то берется текущая дата.
Функция возвращает средние траты в каждый из дней недели за последние три месяца (от переданной даты).
